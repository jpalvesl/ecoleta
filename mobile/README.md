## :clipboard: Description
A mobile application made with React Native

## :rocket: Getting Started

1. Download the repository

  - Using Git
```shell
  git clone https://github.com/jpalvesl/ecoleta.git
```


2. Installation
```shell
  yarn
  npm install
```

3. Execution
```shell
  yarn start
  npm run start
```
